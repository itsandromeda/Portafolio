Special Mention for Tech Innovation in Laboratoria.

This was based on the X Project, a Laboratoria challenge in which we'd have to come up with a concept for a project and write the code using JavaScript events we hadn't seen in class in 48 hours.

Requirements:

  1. Use two events that we had seen in class
  2. Use two new events
  3. Use callbacks

PROCESS

01. Organization

We developed this project in pairs so if we wanted to deliver on time we needed to have a great organization system. We indicated eachother's tasks based on our agility so we wouldn't waste any minute of our time. We used Trello to keep our tasks in order and we set up deadlines to check on our progress and make the changes we needed to make.

02. Research

I came up with this proposal on my search for a social network that focused on sharing travelers experiences online. I made a quick sketch on paper with the structure I had in mind so we could begin with the development process.

03. Development

The selected events we used that we saw in class were:

  1. OnmouseOver
  2. OnmouseOut

The events that we didn't see in class that we used were:

  1. OnmouseMove
  2. DragOver
  3. Drop
